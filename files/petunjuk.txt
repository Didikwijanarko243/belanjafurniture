Ini yang perlu kamu lakukan supaya nyambung semua:
1. Jalankan migration baru (nambah session_id ke tabel wishlists):
bashphp artisan migrate
2. Tambahkan View Composer di app/Providers/AppServiceProvider.php, dalam method boot() — ini yang bikin $cartCount (badge navbar) dan $wishlistProductIds (status hati aktif/tidak) otomatis tersedia di semua view tanpa perlu di-pass manual dari tiap controller:
phpuse Illuminate\Support\Facades\View;
use App\Services\CartService;
use App\Services\WishlistService;

public function boot(): void
{
    View::composer('layouts.app', function ($view) {
        $view->with([
            'cartCount' => app(CartService::class)->getItemCount(),
            'wishlistCount' => app(WishlistService::class)->getCount(),
        ]);
    });

    View::composer(['components.product-card', 'produk.show'], function ($view) {
        $view->with('wishlistProductIds', app(WishlistService::class)->getProductIds());
    });
}
3. Import JS baru di resources/js/app.js:
jsimport { initProductDetailPage } from './product-detail';
import { initWishlistButtons } from './wishlist';

initProductDetailPage();
initWishlistButtons();
